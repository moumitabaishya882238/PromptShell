Why this project exists

Why it is required

PromptShell solves the gap between writing code and understanding it fast, locally, and in a developer-friendly workflow.
Problem → Solution

Problems it addresses

  • Students and beginners often need a quick explanation of compiler errors.
  • Cloud AI tools are not always available, private, or fast enough for local development.
  • Switching between terminal, editor, and browser slows debugging.
  • A resume project should show real engineering: build system, architecture, and tooling.

What PromptShell gives

  • Offline AI assistance using a local model.
  • Integrated run and debug flow for C++ files.
  • Clean, terminal-friendly explanations and fixes.
  • Modular architecture that is easy to extend later.
Simple idea: write code, run PromptShell, see the compiler error, get a clear explanation, and fix the issue without leaving the terminal.
